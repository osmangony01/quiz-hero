Ans : DOM means Document Object Model. It is a programing interface to modify the element from the document.The main purposes for developing the DOM is to maintain a standard programing interface for HTML and XML document.

Ans : Javascript is simply used to modify the content or value of the HTML element on the page. You will see some of the common ways of selecting the elements on a page.
Selecting element by Id - You can select an element based on its unique ID with the getElementById() method. This is the easiest way to find an HTML element in the DOM tree.
Selecting Elements by Class Name - You can use the getElementsByClassName() method to select all the elements having specific class names.
Selecting Elements by Tag Name- You can also select HTML elements by tag name using the getElementsByTagName() method.
Selecting Elements with CSS Selectors - You can use the querySelector() and querySelectorAll() method to select elements that matches the specified CSS selector.

Ans : Bubbling of events is a method of propagation of events in HTML Dom API when the event is in the element inside the other element, and both elements registered in that event. It is a process that begins with an element that activates the event and then bubbles to the elements of the hierarchy. During the events, the event is first recorded and handled in the inner element and then propagates to the outer elements.

Ans : A callback is a function passed as an argument to another function. A callback function can run after another function has finish. We need them because a lot of operations in JavaScript are asynchronous, or put a bit more simply they don't actually stop the program from continuing until they are done like you are probably used to. But instead just run in the background while the rest of the code continues executing.
